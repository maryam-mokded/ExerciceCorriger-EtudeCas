Exercice 4
Enoncé

Ecrire le code permettant de saisir l’âge d’une personne. Et d’afficher le message correspondant.
Si l’âge est inférieur ou égal à zéro, alors afficher un message d’erreur,
Si l’âge est inférieur ou égal à 13, alors afficher "vous êtes encore trop jeune",
Si l’âge est compris entre 13 et 18, alors afficher "Vous êtes encore mineur(e)",
Si l’âge est compris entre 18 et 25, alors afficher "Vous êtes déjà majeur(e)",
Sinon afficher "Ne vous vieillissez donc pas !"

}
}
