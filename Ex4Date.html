Exercice 4
Enoncé

Ecrire le code permettant de remplir un tableau Année contenant les 12 mois de l’année,
et de demander à l’utilisateur de saisir un mois et de chercher sa position
dans le tableau et à quel trimestre il appartient.

Exécution

                               Donner un mois         


code

     let Annee = ( "janvier" , "février" , "mars" , "avril" , "mai" , "juin" , "juillet" , "août" , "septembre" , "octobre" , "novembre" , "décembre" ) ;
     let mois = document . getElementById ( "txt" ) value );
     let i ;
     let c ;

          }
}
   if ( ( c == 1 ) || ( c == 2 ) || ( c == 3 ) ) {

document . getElementById ( "resultat" ) value = ( "Le mois " + mois + " est appartient a la " + t + "trimestre de l’année " );

}
