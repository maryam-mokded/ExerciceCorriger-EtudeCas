Exercice 4
Enoncé

Ecrire le code permettant de Saisir un prix hors taxe, et Calculer et afficher
prix hors taxe et le prix TTC (toute taxe comprise),
sachant que : PrixTTC=prixHT*1.2

Exécution

                                 Donner le prix hors taxe  

                     

code

< script >
Function Calcule (){
     const taux = 20 ;
     let prixHT = Number ( document . fr . prix . value );

     let prixTTC = prixHT + ( prixHT * taux / 100 );
   document . fr . res . value = ( "Le prix hors taxe est" + prixHT +
                  "Alors que le prix TTC est :" + prixTTC );
}
< / script >
